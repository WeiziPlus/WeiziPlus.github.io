    # 公共模块

    • tips:本模块主要是对常用功能进行提取

    • 常用包

      • async 用户异步操作

        • ErrorAsync : 将系统异常异步保存到数据库
        • LogAsync : 配合PC模块和WEB模块，通过拦截器将用户操作异步保存到数据库

      • filter 过滤

        • CorsFilter: 用于配置跨域
          • yml配置文件中通过 global.cors-filter-origins 进行配置
          • 默认为全部 * , 如果配置多个用 , 隔开 (浏览器请求中 Origin 请求头)
        • ParamsFilter: 参数过滤
          • 非空参数自动去除左右空格
          • 使用 Jsoup 进行，html过滤 ,默认 Whitelist.basicWithImages()

      • models generator自动生成的实体类

      • result 返回的响应工具类

      • util 常用的工具类
