*### 表单提交

  • 对element-ui表单进行二次封装

  • props

    • formData : 表单绑定的数据
    • formRules : 表单校验规则
    • formOptions : 表单的组件
      • 示例 :
            [
      {type: 'input', label: '用户名', prop: 'username', required: true},
      {
          type: 'radio', label: '状态', prop: 'status', required: true,
          options: [
              {label: '正常', value: 1},
              {label: '禁用', value: 2}
          ]
      },
      {
          type: 'select', label: '角色', prop: 'roleIds', multiple: true,
          options: []
      },
    ]

      • 参数
        • type : 类型 input输入框、radio单选、select下拉框等,可以根据element-ui自定义
        • label : 显示的当前项的标题
        • prop : 与formData对应
        • required : 必填
        • options : 单选、下列拉框等选项
        • item : 一行显示多个
          • 类型为一个数组，默认flex为1，可以指定
          • station : 占位，默认显示为空

  • $emit

    事件名称 说明 参数
    selectChange 下拉框改变时触发 value, prop
    inputFocus 输入框获得焦点时触发 {$event,item}

  • slot

    名称 说明
    itemHead 放在表单上面部分
    itemTail 放在表单下面部分
