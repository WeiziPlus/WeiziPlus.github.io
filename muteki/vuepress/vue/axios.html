    # axios

    • 对axios进行二次封装,统一处理错误结果和添加加载中动画,并以挂载到Vue实例豪华套餐
      • 参数
        • allUrl:代表当前请求为全路径请求
          • 本次请求只会拦截http状态码不为200的请求，并不对内部返回的内容进行拦截处理
        • allSuccess:代表手动处理请求结果
          • 本次请求只对后端状态码401(登陆过期)进行拦截
        • url:请求的路径，不填写请求域名前缀
        • timeShowLoadAnimation:多长时间之后显示加载中动画,单位毫秒
          • 进行网络请求时一般都很快，如果全都添加加载中动画，那么会有一闪而过的效果，用户体验差，因此加该参数
          • 默认为555(stand by)毫秒,超时之后才会显示加载中动画
        • success为请求的成功回调
          • 只需要处理http状态码为200,同时后端code状态码为200的结果
          • 内置参数为res['data']['data'] 即后端返回的data数据
        • 所有http状态码不为200或者后端code不为200的请求都统一进行弹窗处理
      • 调用实例
            this.$axios({
                        url: '请求的url',
                        method: '请求方式:get请求可以设置该参数',
                        data:{参数:无参数可以不设置该参数},
                        success(data) {
                            console.log(data);
                        }
                    });
