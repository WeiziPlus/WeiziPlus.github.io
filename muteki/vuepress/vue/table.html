    # 表格组件

    • 基于element-ui的表格组件进行的二次封装
      • Complex 复杂表格，多用于进行分页请求的表格

        • 调用该组件必须传递tableDataRequest和tableColumns参数

        • props参数

          • tableDataRequest
            • 示例,url: 请求的路径; data: 请求的额外参数
                  tableDataRequest: {
         url: that.$global.URL.system.sysUser.getPageList,
         data: {}
    }

            • 参数
              • label : 表头
              • prop、fixed、width : element-ui原生属性
              • element : 用于显示element-ui组件，此属性优先级最高,可以自行添加组件
                • 需指定type值
                • element(row) 内置当前行的数据
                • 示例:
                      type: 'tag',
    element({type}) {
        let result = [
            null,
            {content: '查询'},
            {content: '新增', type: 'success'},
            {content: '修改', type: 'warning'},
            {content: '删除', type: 'danger'},
        ];
        return result[type] || {
            content: '未知类型',
            type: 'danger'
        }
    }

                • 用于根据当前行数据动态展示、隐藏按钮，使用时必须判断null == row
          • showSelection : 展示表格左侧多选列
          • showHeaderSearch : 表格顶部右侧输入框，默认请求参数为search,如需变更参数请设置headerSearchColumn

        • 表格方法

          方法名 说明
          renderTable 重新获取表格数据

        • $emit

          事件名 说明 参数
          headerClick 表头点击事件 column
          columnSwitchChange switch值改变触发 {index,prop,row,value}
          cellClick 表格单元格点击事件 {row, column, cell, event}

        • slot

          插槽名 说明
          startColumn 左边的字段列
          endColumn 右边的字段列
      • Simple 简单表格，多用于展示表格样式的数据
        • 调用该组件必须传递tableDataRequest和tableColumns和maxHeight参数
        • maxHeight 请动态计算maxHeight数值(默认为200)，否则表格内容可能显示异常(空白过多)
