    以下指南将帮助你在本地机器上安装和运行该项目，进行开发和测试。关于如何将该项目部署到在线环境，请参考部署小节。

    # 安装要求、步骤

    • 后端环境
      • 安装配置Java环境， JDK1.8
      • 安装 mysql 8 ，创建数据库(utf8mb4、utf8mb4_general_ci)
      • 导入sql /doc/muteki.sql
      • 安装 redis ,下载安装即可
      • 开发工具需要安装 lombok 插件(开发工具推荐IDEA)
      • 检查yml配置文件等相应配置
      • 运行相应的 SpringbootApplication
    • 前端环境
      • 安装配置 node 环境
      • 安装 Vue CLI
      • 进入vue目录执行npm install
      • 运行npm run serve( WebStorm 或者 IDEA 可以直接点击 package.json 文件中第6行"serve": "vue-cli-service serve"左边绿三角)
