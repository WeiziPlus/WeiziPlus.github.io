  • generator 自动生成的实体类
    • 新增单表一条数据 : baseInsert(Object object, boolean getAutoIncrementPrimaryKey)
      • object : 为实体类对象，只会保存有值的字段
      • getAutoIncrementPrimaryKey : 如果需要获取自增id，需指定该参数为 true
    • 新增单表多条数据 : baseInsertList(List<T> list)
      • 该方法会将该表的所有字段进行遍历，并保存到数据库
    • 根据id单表删除 : baseDeleteByClassAndId(Class nowClass, Long/Integer/String id)
      • 根据 id 字段进行删除
    • 根据ids数组单表删除 : baseDeleteByClassAndIds(Class nowClass, T[] ids)
      • 根据 id 字段进行删除,删除 ids 数组的数据 (IN)
    • 单表更新 : baseUpdate(Object object)
      • 根据表的 主键 进行更新操作，只更新有值的字段部分
    • 单表查询一条数据 : baseFindByClassAndId(Class<T> clazz, Long/Integer/String id)
      • 根据 id 字段查询一条数据
    • 单表查询多条数据 : baseFindByClassAndIds(Class<T> clazz, Long/Integer/String[] ids)
      • 根据 id 字段进行查询，查询 ids 数组的数据 (IN)
    • 单表根据字段查询一条数据 : baseFindOneDataByClassAndColumnAndValue(Class<T> clazz, String column, Object object)
      • clazz 实体类 如 : User.class
      • column 自动生成的实体类静态常量 如 : User.COLUMN_USERNAME
      • object 查询的值
            User user = baseFindOneDataByClassAndColumnAndValue(
        User.class,User.COLUMN_USERNAME,"weiziplus");
        1
        2
    • BaseWhere
      • 创建
               BaseWhere<User> baseWhere = new BaseWhere<>(User.class);
        1
      • 添加查询条件 where()
        • 参数为BaseWhereModel或List<BaseWhereModel> baseWhereModelList
        • BaseWhereModel
        • 实例化 BaseWhereModel(String column, BaseWhereEnum where, Object value)
          • column 对应的实体类字段
          • where 条件
          • value 值 (条件为IN或NOT IN时，值为数组类型)
        • 实例化 BaseWhereModel(String column, Object value) 不传条件默认为 =
      • 排序 descOrderBy或ascOrderBy 多个排序条件按照赋值顺序进行排序
    • 单表根据多个字段查询一条数据 : baseFindOneData(BaseWhere<T> baseWhere)
    • 单表获取多条数据 : baseFindListByClassAndColumnAndValue(Class<T> clazz, String column, Object object)
    • 单表根据多个字段查询多条数据 : baseFindList(BaseWhere<T> baseWhere)
    • 单表获取所有数据 : baseFindAllByClass(Class<T> clazz)
