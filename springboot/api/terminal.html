    # CORE/API部分 多终端登录(common/token)

    • 提供了自定义终端类型，登陆过期时间等，单个终端多次登陆会挤掉以前的登录
    • 终端类型 WebTerminalEnum ,枚举
    • 登录示例:
            String token = WebTokenUtils.createToken(WebTerminalEnum.WEB, user.getId(), request, expand);
      1
      第一个参数(WebTerminalEnum.WEB): 终端的类型,指定不同的类型即可实现多终端登录
    • 获取当前登陆用户的userId
      • 示例: (无需关心是哪个终端)
              Long userId = WebTokenUtils.getUserId();
        1
      • 获取当前的终端类型
              WebTerminalEnum terminalEnum = WebTokenUtils.getExpand().getTerminalEnum();
        1
    • 用户退出
      • 退出某个终端
              WebTokenUtils.deleteToken(terminalEnum, userId);
        1
        terminalEnum 终端类型
      • 退出所有终端
              WebTokenUtils.deleteAllToken(userId);
        1
